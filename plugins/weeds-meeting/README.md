# WEEDS 회의록 작성 스킬

WEEDS 회사의 표준 회의록 형식에 맞춰 회의 내용을 정리하고 문서화하는 Claude Code 플러그인입니다.

## 기능

- 회의 정보를 입력받아 회사 표준 형식에 맞춘 회의록 자동 생성
- 참석자, 일시, 장소, 의제, 논의 내용 등을 구조화된 표 형식으로 정리
- 비정형 텍스트(녹취록, 메모)를 분석하여 표준 형식으로 변환

## 사용 방법

다음과 같은 표현으로 스킬을 활성화할 수 있습니다:

```bash
회의록 작성해줘
회의 내용 정리해줘
미팅 기록 만들어줘
```

## 회의록 형식

생성되는 회의록은 다음 정보를 포함합니다:

**필수 정보:**
- 회의 주제 (회의명)
- 일시 (날짜, 시작/종료 시간)
- 주관 부서/팀
- 내부 참석자 (팀, 직급, 이름)
- 회의 장소
- 의제 (논의 주제)
- 회의 내용 및 결과

**선택 정보:**
- 외부 참석자

## 예시

```
회의명 : 20250612 HMAC 관리대장 시스템 설계 및 구현 방안 논의

| ○ 주관부 | AI / PE 팀 |
| ○ 일 시 : | 2025-06-12(목) 13시 00분 ~ 2025-06-12(목) 15시 00분 |
| ○ 장 소 : | 4층 회의실 |
| ○ 내부참석자 : | AI 팀: 김혁 팀장, 손정민 BA<br/>PE 팀: 이인범 BA |
| ○ 외부참석자 | |
| ○ 의 제 : | HMAC 관리대장 시스템 설계 및 구현 방안 논의 |

| ○ 회의결과 | |
| - 내용 | [회의 결과 및 결론 요약] |
```

## 작성 원칙

- **간결성**: 핵심 내용만 명확하게 기술
- **정확성**: 참석자명, 직급, 소속팀 정확히 기재
- **구조화**: 표 형식을 준수하여 가독성 확보
- **요약**: 회의결과는 핵심만 요약하여 작성
